# FUTURE_PE_Task2

# AI Content Marketing using UGC Ads

---

## Problem Statement

Brands struggle to create engaging short-form UGC ads that grab attention within the first 3 seconds and convert viewers into customers.

This project builds a structured AI prompt system to generate high-converting UGC ad scripts using prompt engineering techniques.

---

## Product Selected

Men’s Sunscreen for Oily Skin

Target Audience:
- College students
- Working professionals
- Men aged 18–35

Platform Focus:
- Instagram Reels
- Short-form video ads

---

## Master Prompt Used

You are a performance marketing expert specializing in UGC ads.

Create 5 high-converting Instagram Reel scripts for a men’s sunscreen for oily skin.

Target Audience:
- College students
- Working professionals

Tone:
- Casual
- Relatable
- Scroll-stopping

Each script must include:
1. Hook (first 3 seconds)
2. Problem
3. Solution
4. Benefits
5. Strong CTA

Keep each script under 30 seconds.
Optimize for Instagram Reels.

## Prompt Engineering Strategy

- Assigned AI role as performance marketing expert
- Clearly defined target audience
- Structured script format (Hook → Problem → Solution → Benefits → CTA)
- Added tone control for relatability
- Platform-specific optimization (Instagram Reels)
- Focused on conversion-driven messaging

---

## Key Learnings

- The first 3-second hook determines engagement
- Structured scripts improve clarity and persuasion
- Tone customization increases relatability
- Platform-focused prompts improve relevance
- Clear CTA improves conversion potential
